# CLAUDE.md

## Project Overview

Μαθαίνω ("I learn") is a Progressive Web App (PWA) flashcard app for a young child (age 3.5) learning Greek. Cards show an image or emoji, the Greek word, romanized pronunciation, and English translation. A simple Leitner spaced-repetition system tracks progress per session. Greek is the primary deck; the architecture supports adding other languages (Spanish, etc.) via `data/words.json` without code changes.

## Tech Stack
- **Languages:** HTML, CSS, vanilla JavaScript
- **Libraries:** None — no CDN dependencies
- **Fonts:** Nunito via Google Fonts CDN
- **Build step:** None
- **Python scripts:** Pillow (icons), ElevenLabs + gTTS (audio) — run inside `.venv`

## Key Conventions
- Entry: `index.html` → `src/app.js` + `src/style.css`
- All content lives in `data/words.json` — adding/editing words requires no code changes
- Images go in `images/<category>/` — referenced by `image` field in words.json; emoji is always the fallback
- Audio goes in `audio/` — generated by `scripts/generate_audio.py`
- Python scripts run inside `.venv` — always use `.venv/bin/python`, never global pip

## Development Workflow
```bash
# Serve locally (required — fetch('data/words.json') needs HTTP)
python3 -m http.server 8080
# or
npx serve

# Generate icons (one-time)
python3 -m venv .venv && .venv/bin/pip install Pillow
.venv/bin/python scripts/generate_icons.py

# Generate audio
.venv/bin/pip install gtts elevenlabs
export ELEVENLABS_API_KEY=your_key   # optional, falls back to gTTS
.venv/bin/python scripts/generate_audio.py
```

## Behavior Rules
- No build step — if a library isn't available via CDN, discuss before adding a bundler
- Always use `.venv` for Python — never install packages globally
- When making a structural decision (adding a library, splitting into multiple pages), record it with `/log`
